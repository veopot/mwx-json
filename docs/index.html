Онлайн Каталоги

Manga Watcher X
поддерживает "Онлайн Каталоги".
Таким образом, вам не нужно думать о том, где взять контент для чтения.
Просто выберите желаемый сайт, найдите интересный материал в каталоге и нажмите "Добавить".
Выбранная манга или комиксы появятся в вашей библиотеке.
Все просто, не так ли?

На данный момент есть поддержка свыше 65+ сайтов с общим количеством манги и комиксов больше 500K+! И список продолжает расширятся!


Особенности "Онлайн Каталогов": автоматическое обновление списка доступного материала для чтения и конфигов, возможность авторизации на конкретном сайте, поддержка облачных хранилищ (Dropbox, Google Drive, Yandex.Disk) и возможность добавить в библиотеку уже скачанную мангу с карты памяти.

Репозиторий Парсеров

Manga Watcher X выгодно выделяется на фоне конкурентов своей расширяемостью. Поддержку практически любого сайта можно реализовать с помощью специальных файлов - парсеров. Достаточно скопировать такой файл в папку приложения и все остальное будет сделано за вас. Чтобы еще больше упростить пользователям задачу, был разработан "Репозиторий Парсеров", который является хранилищем для уже готовых парсеров. Достаточно открыть репозиторий из меню "Онлайн Каталоги" и добавить все парсеры, которые вам нужны.



Автоматические Обновления

С Manga Watcher X вам не нужно беспокоиться об обновлениях списка глав манги или комикса. Приложение все сделает за вас и по расписанию. Всем процессом руководят парсеры, поэтому, как только на сайте выйдет новая глава вашей любимой манги или комикса - вы в кратчайшие сроки получите уведомление о наличии обновления.



Одна из лучших Читалок

При разработке Manga Watcher X особое внимание было уделено "Читалке". Доступный функционал: онлайн и оффлайн чтение, поддержка показа номеров страниц, разные типы читалки (облегченный, с анимациями и двустраничный для планшетов), выбор направления чтения и, по желанию, автоопределение направления чтения в зависимости от типа читаемого контента (манга, манхва, комиксы, веб-комиксы), поддержка самых популярных форматов (jpg, png, webp, cbz, zip), множество режимов масштабирования страницы для удобного чтения, большое количество вариантов перелистывания и многое, многое другое!
